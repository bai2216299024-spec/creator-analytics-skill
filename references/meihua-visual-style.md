# 梅花易数内容视觉风格指南

## 账号定位
- 赛道：梅花易数（国学文化科普/传统文化学习/古人观察世界的方法）
- 受众：新手小白 + 进阶学员 + 好奇路人
- 差异化：真实案例积累 + 讲得通俗易懂 + 结合现代生活场景
- 变现：涨粉引流接付费断卦 + 教学课程 + 个人品牌

## 小红书视觉风格（严格遵循，不可擅改）

### 封面/配图
- **风格**：新中式极简 · 纯文字卡片
- **背景**：米白宣纸底 #F5F0E8
- **配色**：青绿色（主标题/重点）、墨黑（正文/副标题）、暗金（装饰/边框）
- **字体**：大字清晰，标题居中
- **装饰**：极简线稿icon、八卦符号、青绿圆点序号、暗金细线边框
- **禁止**：❌ 无人物、❌ 无摄影照片、❌ 无平台水印（如"红红书"）
- **比例**：3:4 竖版
- **卡片类型**：rendered cards（纯文字卡片）或 AI 生图（需在 prompt 中明确"无人物无摄影无水印"）
- **AI prompt 中绝对不能出现**"小红书"等平台名——AI 模型会自行添加水印

### 文案风格
- **开头**：随机切换，不固定单一风格
  - 场景代入型（"昨晚躺床上…"）
  - 提问互动型（"你有没有遇到过…"）
  - 干货直给型（"手机亮屏取象三步走…"）
  - 故事叙述型（"前几天一个朋友问我…"）
- **语气**：像聪明朋友分享发现，不堆术语不说教
- **结构**：步骤化 + 对照表 + 口诀总结
- **互动**：结尾留互动征集钩子引导评论
- **安全边界**：所有个人案例前加声明"不做个人事项判断"
- **收藏引导**：含"建议先收藏""收藏版"等关键词
- **emoji**：适量，每段1-2个

## 常见错误（避免）

### 图片生成
- ❌ 生成含真人照片/人物的 AI 图 → 用户明确要求纯文字卡片
- ❌ 替换整张图来修一个小水印 → 应该原位修复（PIL 覆盖/纹理粘贴）
- ❌ AI prompt 中出现"小红书" → AI 会自行添加"红红书"水印
- ❌ 忽略 prompt_template 中的风格细节 → 米白纸底、青绿墨黑、暗金边框要写全
